# Flowser

![](https://github.com/pilt/flowser/raw/master/gfx/flowser.png)

A high-level interface for [Amazon Simple Workflow][swf] on top of [Boto][boto].

## Requirements

    $ pip install -r requirements.txt


## Usage

[See the documentation.][docs]
